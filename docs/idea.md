# 서비스 이름

TodoMate

## 한 줄 설명

* 이 서비스는 학생 및 직장인 사용자가 해야 할 일을 관리해야 하는 상황에서 할 일을 등록하고 우선순위를 정하여 체계적으로 관리할 수 있게 돕는다.

## 브레인스토밍

### 아이디어 1

* 사용자: 학생, 직장인
* 상황: 해야 할 일이 생겼을 때
* 액션: 사용자가 할 일을 작성한다

### 아이디어 2

* 사용자: 학생, 직장인
* 상황: 할 일을 완료했을 때
* 액션: 사용자가 할 일의 완료 여부를 변경한다

### 아이디어 3

* 사용자: 학생, 직장인
* 상황: 잘못 입력한 내용이 있거나 더 이상 필요 없는 할 일이 있을 때
* 액션: 사용자가 할 일을 수정하거나 삭제한다

## 액션 작성 규칙

* 좋은 예: 사용자가 할 일을 작성한다
* 좋은 예: 사용자가 할 일의 중요도를 설정한다
* 좋은 예: 사용자가 할 일의 완료 여부를 변경한다
